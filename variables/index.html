Variables

What is a variable?

A variable is a bit of code that acts as a stand-in for a value you get later or that you need to use many times

Think of it like a game of mad libs. You pick a few nouns, a verb or two, and an adjective and fill them in throughout a text. The words you choose are variables

What is a data type?

Data types are ways to organize your data. Java is a "Strongly typed" language, so you have to specify what your data is when you declare the variable

When you declare something like

int numBooks = 0

you're creating a variable of the integer datatype with a value of zero

What datatypes exist in Java?

I'm not sure what your class is going to cover, so I'll cover the basics.

There's the integer data type. This can be all whole numbers between -2^31 and 2^31-1.

There's the double data type. This is all numbers, including decimals. There are limits, but you don't care about those right now

Examples:

Integer


                            int five = 3
                            int three = 5
                            System.out.print(three + five) 
                            ...
                            8

double


                            double pie = 3.14
                            double taurus = 6.28
                            System.out.print(pie + taurus)
                            ...
                            9.42

String

String data types are used whenever you need text or something in text form. You'll use a lot of these, some make sure you're familliar with them. When typing in a string and using them in output statements, make sure you're familliar with where your spaces need to be. "thing " and "thing" will have differing results in output statements.

String
